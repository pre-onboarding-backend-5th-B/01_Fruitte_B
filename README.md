# 기능
## 목록
1. 회원 가입 및 로그인
2. 상품 CRUD
3. 결제 관리
4. 주문 내역
## 제약사항 
- 유저는 이용자, 관리자로 나누어져 있음
- 상품 및 결재, 주문은 관리자만 입력, 수정, 삭제 가능
- 이용자는 회원가입, 결재, 주문을 제외하고 조회 권한만 허용
